# edukasi-python

Python Learning Portal adalah website edukasi Python berbasis React JS, Vite, Tailwind CSS, React Router DOM, dan Lucide React.

## Menjalankan Project

```bash
npm install
npm run dev
```

## Build Production

```bash
npm run build
```

Project ini tidak memakai login, backend, atau database. Semua konten menggunakan data dummy JavaScript.
